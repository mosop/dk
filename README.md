# Dk

Yet another Crystal library for accessing the Docker API.

Experimental.

[![Build Status](https://travis-ci.org/mosop/dk.svg?branch=master)](https://travis-ci.org/mosop/dk)

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  gh:
    github: mosop/dk
```
